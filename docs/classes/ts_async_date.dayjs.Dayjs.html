diff

  • diff(date?: null | string | number | Date | dayjs.Dayjs, unit?: "s" | "millisecond" | "second" | "minute" | "hour" | "day" | "month" | "year" | "date" | "milliseconds" | "seconds" | "minutes" | "hours" | "days" | "months" | "years" | "dates" | "d" | "M" | "y" | "h" | "m" | "ms" | "week" | "weeks" | "w" | "quarter" | "quarters" | "Q", float?: boolean): number
    • Defined in node_modules/dayjs/index.d.ts:303

    This indicates the difference between two date-time in the specified unit.

    To get the difference in milliseconds, use dayjs#diff

    const date1 = dayjs('2019-01-25')
    const date2 = dayjs('2018-06-05')
    date1.diff(date2) // 20214000000 default milliseconds
    date1.diff() // milliseconds to current time

    To get the difference in another unit of measurement, pass that measurement as the second argument.

    const date1 = dayjs('2019-01-25')
    date1.diff('2018-06-05', 'month') // 7

    Units are case insensitive, and support plural and short forms.

    Docs: https://day.js.org/docs/en/display/difference

    Parameters

    • Optional date: null | string | number | Date | dayjs.Dayjs
    • Optional unit: "s" | "millisecond" | "second" | "minute" | "hour" | "day" | "month" | "year" | "date" | "milliseconds" | "seconds" | "minutes" | "hours" | "days" | "months" | "years" | "dates" | "d" | "M" | "y" | "h" | "m" | "ms" | "week" | "weeks" | "w" | "quarter" | "quarters" | "Q"
    • Optional float: boolean

    Returns number
